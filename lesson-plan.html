☁ GitHub Sync

Connect your GitHub account so every save pushes changes to the shared repo. Changes load automatically for all team members when they open the page.

How to get a token:
1. GitHub.com → Settings → Developer settings
2. Personal access tokens → Fine-grained tokens → Generate new token
3. Repository access: Only select repositories → curriculum-planning-tool
4. Repository permissions: Contents → Read and Write
5. Generate token → copy & paste it above
